Timestamp Microservice

By

Dhanushu Uzumaki

User Stories

I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: November 30, 1995)

If it does, it returns both the Unix timestamp and the natural language form of that date.

If it does not contain a date or Unix timestamp, it returns null for those properties.

Usage

https://fcc-api-basejumps.herokuapp.com/api/timestamp/nov-30-1995
https://fcc-api-basejumps.herokuapp.com/api/timestamp/817669800

Response

{"unix":817669800,"natural":"November 30, 1995"}
